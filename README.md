```
helps for wty(PowerShell)

$env:JAVA_HOME = "D:\Java\jdk-17"

2. 将 JDK 17 的 bin 目录放到 PATH 最前面
$env:PATH = "$env:JAVA_HOME\bin;$env:PATH"

验证切换是否成功（输出应为 Java 17 版本）
java -version

./gradlew assembleRelease

```
